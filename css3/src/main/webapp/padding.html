padding

요소의 내부(테두리 기준) 여백

1. padding의 방향별 지정 속성

  • padding-top:n; 내부 위쪽에만 n적용
  • padding-right:n; 내부 오른쪽에만 n적용
  • padding-bottom:n; 내부 아랫쪽에만 n적용
  • padding-left:n; 내부 왼쪽에만 n적용

2. padding의 값 갯수별 적용법

  • padding: n; 양사방 동시에 n적용
  • padding: n n; 상하, 좌우가 서로 같은 값
  • padding: n n n; 상, 좌우 동일, 하 순서
  • padding: n n n n; 상, 우, 하, 좌의 순서
