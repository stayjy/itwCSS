  • background-image:url(" "); 배경 이미지

background 배경

  • background-color: ; 배경색상
  • background-image:url(""); 배경이미지

background-image의 세부속성

  • background-repeat: ; 배경이미지 반복 지정
    • repeat 기본값. 양사방 반복
    • repeat-x x축(가로)으로만 반복
    • repeat-y y축(세로)으로만 반복
    • no-repeat 반복 안함(한번만 나옴)
  • background-size: ; 배경이미지 크기 기정
    • cover 이미지 하나로 영역전체를 다덮기
      (이미지가 잘려나갈 수 있음)
    • contain 이미지 하나로 가로나 세로 중 한쪽에 맞춰서 최대한 덮기
      (이미지가 잘려나가지 않음)
    • Xn Yn 가로 세로 순으로 직접 지정(npx,n%,nem,...)
  • background-position: ; 배경이미지 위치 지정
    • left/center/right top/center/bottom 수평 수직 순으로 조합
    • Xn Yn 가로 세로 순으로 직접 지정(npx,n%,nem,...)
  • background-attachment: ; 배경이미지 스크롤 여부 지정
    • scroll - 브라우저 스크롤바에는 스크롤, 자신영역의 스크롤바에는 고정 됨
    • fixed - 브라우저와 자신 영역의 스크롤바 모두 고정 됨
    • local - 브라우저와 자신 영역의 스크롤바 모두 스크롤 됨.

background-attachment: ; 배경이미지 스크롤 여부 지정

fixed - 브라우저와 자신 영역의 스크롤바 모두 고정 됨
local - 브라우저와 자신 영역의 스크롤바 모두 스크롤 됨.
