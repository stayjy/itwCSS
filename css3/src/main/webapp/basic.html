CSS basic

태그가 화면에 나타나는 방식(스타일)을 지시하는 언어.

1. CSS 적용 법(방식)

  • External CSS - 외부 파일 방식.
    스타일소스를 독립된 파일(*.css)로 만들고 웹문서와 연결시켜 적음.
    하나의 스타일파일로 여러 웹문서를 동시에 같은 스타일로 적용하거나 수정 가능.
  • Internal CSS - 내장 방식
    스타일소스를 웹문서의 head영역에 적용.
    해당 웹문서에만 적용됨.
  • Inline CSS - 인라인 방식
    웹문서의 body영역의 해당 태그에 직접 적용
    소스가 길어지고 통일성 있는 스타일 구성이 불가하며 유지보수에도 불리.
    세가지 방식중 CSS적용 우선순위가 가장 높음.
    작업 환경상 body영역만 수정 가능한 상황일 때 사용.

2. CSS 문법

선택자 { 속성:속성값; }

3. CSS 선택자 종류 대표 3가지

  • 선택자 - 태그 그대로 선택자로 활용.
    자동 적용, 강제 적용.
  • id선택자 - 태그 뒤에 id를 추가하고 #id를 불러와서 선택자로 활용.
    해당 스타일내용을 중복적용 불가(id값이 유일한 값)하므로 주로 구조적인 용도로 쓰임.
  • class선택자 - 태그 뒤에 class를 추가하고 .class를 불러와서 선택자로 활용.
    해당 스타일내용을 중복적용 가능(class값은 복수 적용)하므로 주로 디자인적인 용도로 쓰임.
