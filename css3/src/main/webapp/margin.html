margin

요소의 부(테두리 기준) 여백

margin의 방향별 지정 속성

  • margin-top:n;
  • margin-right:n;
  • margin-bottom:n;
  • margin-left:n;

margin값 갯수별 적용법

  • margin:n ;
  • margin:n n ;
  • margin-n n n ;
  • margin-n n n n;

항상 요소가 수평 중앙에 위치

= 좌우 margin을 항상 같게(auto) 함 margin-left: auto; margin-right: auto; 또는 margin:n auto;

inline 요소 margin

inline(한줄에 나란히오는)요소는 상하 margin은 적용 불가, 좌우 margin은 인접요소마다 각각 적용
